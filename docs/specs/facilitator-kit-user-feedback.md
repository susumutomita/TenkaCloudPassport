# Facilitator Kit First-time User Feedback

## Scope and status

- Issue: <https://github.com/susumutomita/TenkaCloudPassport/issues/27>。
- Role: Product を初めて使う参加者と初めて進行する Local Champion の机上 Simulation。
- Final document verdict: `ALLOW`。
- Physical Dry Run: `Not run`。

## Feedback integration

| First-time confusion | Final state | Resolution |
| --- | --- | --- |
| Setup と 20 分の開始点が重複する。 | Resolved | Setup は 1 回、最初の Invite 生成から最大 20 分にした。 |
| Product と Research を同時に断る質問に見える。 | Resolved | Product Preview と Research Script を別の判断にした。 |
| 退出すると他の端末を待つように見える。 | Resolved | 退出者の Data だけを直ちに破棄し、Host Loss と分けた。 |
| Host と Participant 数、必要端末が分からない。 | Resolved | Host を含む合計 2〜6 名、1 人 1 台と明記した。 |
| 同じ QR を全員で Scan できるように見える。 | Resolved | 1 名ごとの fresh Invite と Secret Rotation を明記した。 |
| English だけでは開催前判断ができない。 | Resolved | English Support Matrix と対訳 Record を追加した。 |
| 終了、未開始、停止の違いが色に依存する。 | Resolved | 3 つの文字 Label と読み上げ文を追加した。 |

## Remaining first-time evidence

実在する未経験者が、10 分以内の説明だけで Kit を見つけ、`P0`〜`P10` と `R1`〜`R10` を判断できるかは
`Not run` と記録する。実機経路も `Not run` のため、この机上 Feedback を参加者体験や Field Ready の証拠に
使わない。迷いが 1 件でもあれば、個人情報や発言を記録せず、分類件数だけで Kit を改訂して再実施する。
