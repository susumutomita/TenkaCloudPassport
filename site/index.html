Not a business card

これは、何ではないか。

NOT / 01

名刺管理アプリではない

連絡先を集めません。氏名・メール・電話・会社名を入力する欄がそもそもありません。

NOT / 02

SNS ではない

アカウントもフォローもありません。Lounge が終われば、残るのはあなたの記憶と会話だけです。

NOT / 03

マッチングではない

スコアで人を並べません。Pet が見つけるのは「最初のひと言」の橋を 1 つだけです。

How it works

Pet が橋を架け、人間が渡る。

  1. Pet を準備する

    アカウント登録なし。Pet の名前と絵文字、そして同梱カタログから選んだ「会話の材料」（Topics / Offer / Looking For）だけ。入力は明示保存するまで端末に残りません。

  2. QR で 20 分の Lounge に入る

    Host が表示する使い捨ての招待 QR を読み取り、共有内容の Preview に明示同意してから参加します。Lounge は 20 分で必ず終わります。

  3. Pet 同士が Bridge を 1 つ見つける

    公開を許可した手掛かりだけを突き合わせ、「この 2 人ならこの話題から始められる」という橋を 1 つ提案します。

  4. アプリは退く

    会話が始まったら、アプリの役目は終わりです。Lounge のデータは退出・Host 終了・20 分満了のいずれか最も早い契機で破棄されます。

Privacy by design

預からない、が設計です。

  • アカウント不要登録もログインもありません。サーバーにあなたのデータはありません。
  • 端末内で完結Profile は端末の中だけ。持ち出しは自分で行う JSON Backup のみです。
  • 20 分で破棄Lounge 由来のデータは永続化せず、最早の契機で破棄します。
  • テレメトリなし収集も送信もしません。AI 判定も端末内で完結する設計です。現在は Rules Provider が担い、Local LLM は Planned（default branch 未収載）です。

Try it

手元で動かす。

git clone https://github.com/susumutomita/TenkaCloudPassport.git
cd TenkaCloudPassport
make install
make start   # スマホの Expo Go で QR を読む

スマホに Expo Go を install し、Mac と同じ Network につないでから、Terminal に表示される QR を読み取ってください。Browser で試す場合は bun run web です。
