LT の後に話しかけてもらえたのに、名刺を作っていなかった。有料の名刺サービスに お金は払いたくない。TenkaCloud Passport は、名前・肩書き・自己紹介を QR で 見せるだけの、無料でアカウント不要のアプリです。相手はアプリを入れず、標準 カメラで QR を読み取るだけで自己紹介ページが開きます。連絡先への追加は、 そのページ内で相手が選んだときだけの任意操作です。

これは、何ではないか。

NOT / 01

有料サブスクではない

月額課金の名刺サービスではありません。無料で、アカウントの有効期限もありません。

NOT / 02

アカウント登録はない

サインアップもログインもありません。名前を入力したその場で、すぐ使えます。

NOT / 03

データを預からない

自己紹介カードは端末の中だけに保存します。サーバーには何も送信しません。

How it works

渡すのは QR コード 1 枚だけ。

  1. 自己紹介カードを作る

    名前（必須）・肩書き・所属・自己紹介・リンクを入力します。入力は明示保存するまで端末に残りません。

  2. QR を見せる

    保存すると、名前・自己紹介と大きな実 QR コードが画面に表示されます。この画面を相手に見せるだけです。

  3. 相手が標準カメラで読み取る

    アプリのインストールは不要です。iPhone / Android の標準カメラで QR を読み取ると、ブラウザで自己紹介ページが開きます（相手がオンラインである必要があります）。連絡先への追加は、そのページ内のボタンを押した場合だけの任意操作です。

  4. 会話が続く

    自己紹介を読んだ相手から、次の話が始まります。名刺交換の代わりに、会話のきっかけを渡します。

Roadmap

Step 1 は実装済み。その先は構想です。

  1. 実装済み

    自己紹介カードを QR で渡す

    相手はアプリ不要、標準カメラで自己紹介ページが開きます。連絡先への追加はページ内で相手が選ぶ任意操作です。本ページが説明している機能です。

  2. 構想

    受け取った相手をリサーチする

    受け取った側のアプリが、相手をインターネットで自動リサーチしてオントロジー（人物・活動・関心の構造）を作る構想です。「外部と通信しない」という現行方針の転換を伴うため、着手時に ADR で opt-in 設計を定めます。

  3. 構想

    近い人を可視化する

    イベント参加者の中で「やっていることが近い人」を可視化する構想です。

  4. 構想

    端末内エージェント同士が接点を見つける

    相手もアプリを入れたら、端末内 LLM エージェント同士が自己紹介を交換して次の打ち合わせにつながる話題を見つける構想です。既存の Pet / Lounge / Bridge の資産がここで主役に戻る想定です。

Step 2〜4 はまだ実装していません。「できます」とは書かず、構想として区別して掲載しています。

Privacy by design

預からない、が設計です。

  • アカウント不要登録もログインもありません。サーバーにあなたのデータはありません。
  • 端末内で完結自己紹介カードは端末の中だけ。持ち出しは自分で行う JSON Backup のみです。
  • 共有は QR を見せる操作だけ相手の情報を受け取る経路は持ちません。渡すのは自分のカードだけです。
  • テレメトリなし収集も送信もしません。サーバーもアカウントも要求しない設計です。

Try it

手元で動かす。

git clone https://github.com/susumutomita/TenkaCloudPassport.git
cd TenkaCloudPassport
make install
make start   # スマホの Expo Go で QR を読む

スマホに Expo Go を install し、Mac と同じ Network につないでから、Terminal に表示される QR を読み取ってください。Browser で試す場合は bun run web です。

開発環境なしでスマホから試したい場合は、 Web 版 を Browser で直接開いてください。データは端末内ストレージに保存するため、Browser や 端末を変えるとカードは共有されません。初回表示にはオンライン接続が必要です。iOS Safari は共有ボタンから「ホーム画面に追加」を、Android Chrome はメニューから「ホーム画面に追加」 または「アプリをインストール」を選ぶと、ホーム画面のアイコンからアプリのように起動できます （App Store 等のストア経由の配布ではありません）。
